open-archaeo

a list of open source archaeological software and resources

how to contribute:

  • suggest new items to be included on the list
  • brainstorm ideas for new tools
  • suggest alternative categorization schemes
  • report bugs, suggest new features

get in touch!

  • by forking and submitting a pull request
  • by opening an issue on github
  • by getting in touch via email or twitter
  • by using this handy webform

contact info:

  • email: z [dot] batist [at] mail [dot] utoronto [dot] ca
  • twitter: @mtl_zack
